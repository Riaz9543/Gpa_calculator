# 🎓 Complete SSC GPA Calculator for Bangladesh

**Version 2.0** - A comprehensive, accurate, and user-friendly SSC GPA calculator built for the Bangladesh education system.

## ✨ Features

### 🎯 **Accurate Calculation Engine**
- **Official SSC Grading System**: Uses the exact grading scale used by Bangladesh education boards
- **Proper ICT Handling**: Mandatory ICT subject calculation with failure detection
- **Optional Subject Bonus**: Correctly calculates bonus points for 4th subject (grade - 2.0)
- **Failure Detection**: Any F grade results in overall GPA of 0.00
- **GPA Capping**: Maximum GPA is properly capped at 5.00

### 📚 **Complete Subject Support**
- **Science Group**: Physics, Chemistry, Biology, Mathematics, and all core subjects
- **Humanities Group**: History, Civics, Economics, Social Science, and core subjects  
- **Commerce Group**: Accounting, Business Entrepreneurship, Finance & Banking, and core subjects
- **Core Subjects**: Bangla, English, Mathematics, Religion for all groups
- **Additional Subjects**: ICT (mandatory) and Optional 4th subject

### 🚀 **Advanced User Experience**
- **Modern UI/UX**: Beautiful, responsive design with smooth animations
- **Keyboard Shortcuts**: 
  - Enter: Move to next field
  - Ctrl+Enter: Calculate GPA
  - ESC: Reset calculator
- **Auto-Save**: Automatically saves your progress in browser storage
- **Comprehensive Results**: Shows GPA, letter grade, and division equivalent
- **Input Validation**: Real-time validation with helpful error messages

### 📊 **Official Grading System**

| Marks Range | Grade | Points | Description |
|-------------|-------|--------|-------------|
| 80-100      | A+    | 5.00   | Excellent   |
| 70-79       | A     | 4.00   | Very Good   |
| 60-69       | A-    | 3.50   | Good        |
| 50-59       | B     | 3.00   | Average     |
| 40-49       | C     | 2.00   | Below Average |
| 33-39       | D     | 1.00   | Poor        |
| 0-32        | F     | 0.00   | Fail        |

### 🔢 **GPA Calculation Formula**

```
Total Points = Sum of all main subject points + ICT points
Optional Bonus = Max(0, Optional Grade - 2.0)
Final GPA = Min(5.00, (Total Points + Optional Bonus) / Number of Main Subjects)
```

## 📖 **How to Use**

1. **Select Your Group**: Choose from Science, Humanities, or Commerce
2. **Enter Grades**: Select grades for all main subjects (required)
3. **Add ICT Grade**: Enter your ICT subject grade (mandatory)
4. **Optional Subject**: Add 4th subject grade for bonus points (optional)
5. **Calculate**: Click calculate or press Ctrl+Enter
6. **View Results**: See your GPA, grade, and division equivalent

### **Important Rules**

- Optional subject adds bonus only if > 2.0
- Maximum GPA is capped at 5.00
